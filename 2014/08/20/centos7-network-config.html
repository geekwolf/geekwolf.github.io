关闭不必要的服务

最小化安装的Centos7系统并没有nano、vim、wget、curl、ifconfig、lsof命令，这里首先安装一下：
yum -y install nano vim wget curl net-tools lsof
可以通过netstat和lsof查看系统都运行了哪些服务，将不必要的进行关闭
